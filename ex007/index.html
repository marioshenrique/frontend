Adicionar Imagens ao HTML

Adicionar imagens a umam página HTML é um dos elementos essenciair para criar uma interface visualmente atraente. Isso é feito usando a tag img, que é uma tag autocontida e permite a inserção de imagens de diferentes formatos em seu site. A seguir, explicamos como utilizar essa tag e seus principaus atributos.

Sintaxe Básica da Tag <img>

Abaixo temos uma imagem para ilustrar a forma mais básica de adicionar uma imagem em um arquivo HTML.

A imagem abaixo tem largura de 400 pixeis. A altura é definida automaticamente, calculada para manter a proporção real da imagem sem causar distorções.

Basicamente são fornecidos dois atributos:

  • Atributo 'src': define o caminho do arquivo de imagem que será exibido na página.
  • Atributo 'alt': fornece uma descrição da imagem que será exibida se a a imagem não puder ser carregada.

Imagens Responsivas

Para garantir que sua imagem se ajuste a diferentes tamanhos de tela (como desktops, tablets e smartphones), é recomendável usar CSS para definir uma imagem responsiva.

Com isso, a imagem se adaptará ao tamanho da tela, sendo redimensionada de forma fluida para diferentes dispositivos.
